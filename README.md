# Human Development Index (HDI) Prediction

## Project Overview

This project predicts the Human Development Index (HDI) of a country using Machine Learning.

The model uses:

- Country
- Expected Years of Schooling
- Mean Years of Schooling
- Gross National Income (GNI) Per Capita

to predict the HDI score and classify it into:

- Low HDI
- Medium HDI
- High HDI
- Very High HDI
